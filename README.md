# ProductHuntViewer
Test task for Glads Group.
## Features
* Configuration changes support.
* Dagger 2, Retrofit 2, ButterKnife, Glide.
* Notification service works after device being restarted.
* Unit and UI tests.
* Animated RecyclerView.
* Cache.
